# Gulistan App - Madrasa Attendance Management System

A modern web application for managing madrasa attendance tracking built with React and Vite.

## Features

- Student and Teacher authentication
- QR code scanning for attendance
- Attendance tracking and management
- Bilingual support (English/Urdu)
- Responsive mobile-first design

## Tech Stack

- React 18
- Vite
- Lucide React Icons
- IndexedDB for local storage

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

The app will run on `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This app is deployed on Render at: https://gulistan-app.onrender.com/
